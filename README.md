Programación Orientada a Objetos en Python

Este repositorio contiene los trabajos y ejercicios desarrollados en la asignatura Programación Orientada a Objetos (POO), enfocados en la aplicación práctica de los conceptos fundamentales de la programación en Python.

Durante el desarrollo de los proyectos se abordan temas como:

Uso correcto de tipos de datos (enteros, flotantes, cadenas, booleanos y estructuras compuestas).

Aplicación de identificadores descriptivos y convenciones de nomenclatura según el estándar snake_case.

Implementación de programas utilizando Programación Orientada a Objetos, mediante la creación de clases, atributos y métodos.

Modelado de situaciones del mundo real a través de objetos, promoviendo la modularidad, reutilización y claridad del código.

Buenas prácticas de programación, incluyendo comentarios, documentación y organización del código.

Uso de herramientas de desarrollo como PyCharm y control de versiones con GitHub.

El objetivo principal de este repositorio es reforzar la comprensión de los principios de la POO y desarrollar soluciones claras, estructuradas y funcionales, fortaleciendo las habilidades fundamentales en programación con Python.
